From this page, you can use the following links to watch the log entries for this system as it's running, which can be useful for debugging issues.

  • OpenTrickler log This monitors just the OpenTrickler code that runs on this Raspberry Pi and nothing else.
  • Bluetooth log This monitors the OpenTrickler Bluetooth code that runs on this Raspberry Pi and nothing else.
  • Raspberry Pi System log This will let you see if the system is booting up properly, and if anything has crashed, among other things.
